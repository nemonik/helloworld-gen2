# GoLang Helloworld

A simple helloworld GoLang app for my <https://github.com/nemonik/hands-on-DevOps-gen2>

### Prerequisites

GoLang and bunch more tools must be installed and configured.

### Testing

Typcially

```bash
make test
```

Alternatively

```bash
make sonar
```

### Running

```bash
make run
```
